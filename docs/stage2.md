## Day 11: 导数与偏导数

**一句话**：导数度量函数的变化率，偏导数是多变量函数的"单变量导数"。

### 核心定义

- **导数**：f'(x) = lim(h→0) [f(x+h) - f(x)] / h
- **几何意义**：函数曲线在某点的切线斜率
- **偏导数**：∂f/∂xᵢ，固定其他变量，只对 xᵢ 求导
- **梯度向量**：所有一阶偏导数组成的向量

### AI 中的应用

- **反向传播**：计算损失函数对每个权重的偏导数
- **激活函数求导**：ReLU、sigmoid、tanh 的导数
- **学习率**：梯度乘以学习率决定更新步长

### 关键公式

<div class="formula-box">

```
常用导数：
d(xⁿ)/dx = nxⁿ⁻¹
d(eˣ)/dx = eˣ
d(ln x)/dx = 1/x
d(sigmoid(x))/dx = σ(x)(1-σ(x))
d(ReLU(x))/dx = 1 if x>0 else 0

链式法则 (单变量)：dy/dx = dy/du × du/dx
```

</div>

## Day 12: 梯度

**一句话**：梯度指向函数增长最快的方向，是优化的指南针。

### 核心定义

- **梯度**：∇f = [∂f/∂x₁, ∂f/∂x₂, ..., ∂f/∂xₙ]ᵀ
- **梯度方向**：函数值增长最快的方向
- **负梯度方向**：函数值下降最快的方向

### AI 中的应用

- **梯度下降**：w = w - η∇L(w)
- **反向传播**：高效计算神经网络梯度
- **梯度裁剪**：防止梯度爆炸

### 关键公式

<div class="formula-box">

```
梯度定义：∇f(x) = [∂f/∂x₁, ..., ∂f/∂xₙ]ᵀ
方向导数：Dᵥf = ∇f · v (v 是单位向量)
梯度下降：wₜ₊₁ = wₜ - η∇L(wₜ)
```

</div>

## Day 13: 链式法则

**一句话**：链式法则是反向传播的核心，让深层网络的梯度计算成为可能。

### 核心定义

- **链式法则**：复合函数的导数 = 各层导数的乘积
- **多变量**：∂y/∂x = Σ(∂y/∂uᵢ × ∂uᵢ/∂x)
- **计算图**：将复合函数分解为基本操作

### AI 中的应用

- **反向传播算法**：从输出层向输入层逐层计算梯度
- **自动微分**：框架自动应用链式法则

### 关键公式

<div class="formula-box">

```
神经网络反向传播：
δᴸ = ∂L/∂aᴸ (输出层误差)
δˡ = (Wˡ⁺¹)ᵀδˡ⁺¹ ⊙ σ'(zˡ) (误差反向传播)
∂L/∂Wˡ = δˡ(aˡ⁻¹)ᵀ (权重梯度)
```

</div>

## Day 14: 泰勒展开

**一句话**：用多项式逼近复杂函数，是优化算法的理论基础。

### 核心定义

- **泰勒展开**：f(x) = Σ(f⁽ⁿ⁾(a)/n!)(x-a)ⁿ
- **一阶近似**：f(x) ≈ f(a) + f'(a)(x-a)
- **二阶近似**：f(x) ≈ f(a) + f'(a)(x-a) + (1/2)f''(a)(x-a)²

### AI 中的应用

- **梯度下降**：基于一阶泰勒展开
- **牛顿法**：基于二阶泰勒展开

### 关键公式

<div class="formula-box">

```
一阶泰勒展开：f(x) ≈ f(a) + ∇f(a)ᵀ(x-a)
二阶泰勒展开：f(x) ≈ f(a) + ∇f(a)ᵀ(x-a) + (1/2)(x-a)ᵀH(a)(x-a)
Hessian 矩阵：Hᵢⱼ = ∂²f/∂xᵢ∂xⱼ
```

</div>

## Day 15: 梯度下降法

**一句话**：最基础的优化算法，沿着负梯度方向迭代更新参数。

### 核心定义

- **梯度下降**：wₜ₊₁ = wₜ - η∇L(wₜ)
- **学习率 η**：步长大小
- **BGD**：每次用全部数据
- **SGD**：每次用一个样本
- **Mini-batch**：折中方案

### 关键公式

<div class="formula-box">

```
批量梯度下降：wₜ₊₁ = wₜ - η∇L(wₜ)
小批量：wₜ₊₁ = wₜ - η∇L(wₜ; Bₜ)
```

</div>
