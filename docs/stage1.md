## Day 1: 向量与向量空间

**一句话**：向量是 AI 表示一切数据的基本单元。

### 核心定义

- **向量**：有序的数字数组，既有大小又有方向
- **向量空间**：满足加法和数乘封闭的向量集合
- **基向量**：张成向量空间的最小线性无关向量组

### 几何直观

<div class="callout callout-tip">

- 2D 向量是平面上的箭头，3D 向量是空间中的箭头
- 高维向量虽然无法可视化，但数学性质完全相同
- 向量的每个维度可以看作一个"特征"

</div>

### AI 中的应用

- **词向量** (Word2Vec, GloVe)：将词语映射为稠密向量
- **图像表示**：图片 = 像素向量或特征向量
- **用户画像**：用户特征 = 高维向量
- **注意力机制**：Query/Key/Value 都是向量

### 关键公式

<div class="formula-box">

```
点积：a · b = Σ(aᵢ × bᵢ) = |a||b|cos(θ)
模长：||a|| = √(a · a)
余弦相似度：cos(θ) = (a · b) / (||a|| × ||b||)
```

</div>

### 思考题

为什么余弦相似度比欧氏距离更适合衡量语义相似性？

<details>
<summary>点击查看答案提示</summary>

余弦相似度只关注方向差异，不受向量长度影响。在文本等高维稀疏数据中，长度往往代表文档长度等无关因素，而方向（角度）更能反映语义相似性。

</details>

## Day 2: 矩阵与矩阵运算

**一句话**：矩阵是向量的批量变换，是神经网络的骨架。

### 核心定义

- **矩阵**：m×n 的数字阵列，可看作多个向量的组合
- **矩阵乘法**：A(m×n) × B(n×p) = C(m×p)，要求内维相等
- **转置**：Aᵀ，行列互换，(Aᵀ)ᵢⱼ = Aⱼᵢ

### 几何直观

<div class="callout callout-warning">

- 矩阵乘法 = 线性变换的组合
- 矩阵的每一列是基向量变换后的结果
- 矩阵乘法不满足交换律：AB ≠ BA（一般情况）

</div>

### AI 中的应用

- **全连接层**：输出 = W × 输入 + b，W 是权重矩阵
- **卷积层**：卷积核是小矩阵，在图像上滑动
- **Batch 处理**：一次处理多个样本，矩阵第一维是 batch size
- **Transformer**：Q/K/V 矩阵并行计算所有位置的注意力

### 关键公式

<div class="formula-box">

```
矩阵乘法：(AB)ᵢⱼ = Σₖ(Aᵢₖ × Bₖⱼ)
转置性质：(AB)ᵀ = BᵀAᵀ
单位矩阵：I × A = A × I = A
```

</div>

### 思考题

为什么神经网络中权重矩阵的维度是 (输出维度 × 输入维度)？

## Day 3: 线性变换

**一句话**：矩阵乘法的本质是对空间进行变换。

### 核心定义

- **线性变换**：保持向量加法和数乘的变换 T
  - 性质 1：T(u + v) = T(u) + T(v)
  - 性质 2：T(cv) = c × T(v)
- 任何线性变换都可以用矩阵表示

### 几何直观

- **旋转**：矩阵乘法可以旋转向量
- **缩放**：对角矩阵实现各维度独立缩放
- **剪切**：平行四边形变形
- **投影**：高维空间投影到低维

### AI 中的应用

- **神经网络层** = 线性变换 + 非线性激活
- **降维**：PCA 是将数据投影到主成分方向
- **注意力**：Attention(Q,K,V) = softmax(QKᵀ/√d)V，本质是加权投影
- **数据增强**：旋转、缩放、剪切图像

### 关键公式

<div class="formula-box">

```
旋转矩阵 (2D, θ角)：
[cos(θ)  -sin(θ)]
[sin(θ)   cos(θ)]

缩放矩阵：
[sₓ  0 ]
[0   sᵧ]
```

</div>

### 思考题

为什么神经网络需要非线性激活函数？只有线性变换会怎样？

## Day 4: 行列式

**一句话**：行列式度量线性变换对空间的"缩放因子"。

### 核心定义

- **行列式**：det(A) 或 |A|，方阵的标量值
- **几何意义**：变换后空间体积的缩放比例
- **det(A) = 0**：矩阵不可逆，空间被压缩到低维
- **det(A) < 0**：变换包含镜像翻转

### 几何直观

| 维度 | 行列式含义 |
|------|-----------|
| 2D | 变换后平行四边形的面积 |
| 3D | 变换后平行六面体的体积 |
| det = 0 | 空间被压扁（信息丢失） |
| \|det\| > 1 | 空间膨胀 |
| \|det\| < 1 | 空间收缩 |

### AI 中的应用

- **可逆性判断**：det ≠ 0 是矩阵可逆的充要条件
- **归一化流** (Normalizing Flow)：用行列式计算概率密度变换
- **雅可比行列式**：变量替换时的概率密度调整
- **特征值乘积**：det(A) = λ₁ × λ₂ × ... × λₙ

### 关键公式

<div class="formula-box">

```
2×2 行列式：|A| = ad - bc  (A = [[a,b],[c,d]])
性质：det(AB) = det(A) × det(B)
逆矩阵行列式：det(A⁻¹) = 1/det(A)
```

</div>

### 思考题

为什么神经网络中要避免权重矩阵的行列式接近 0？

## Day 5: 逆矩阵与伪逆

**一句话**：逆矩阵是变换的"撤销操作"，伪逆是不可逆时的最佳近似。

### 核心定义

- **逆矩阵**：A⁻¹ × A = A × A⁻¹ = I
- **可逆条件**：方阵且 det(A) ≠ 0（满秩）
- **伪逆 (Moore-Penrose)**：A⁺，对任意矩阵都有定义
- **最小二乘解**：x = A⁺b 最小化 ||Ax - b||²

### AI 中的应用

- **线性回归解析解**：w = (XᵀX)⁻¹Xᵀy
- **推荐系统**：矩阵分解求伪逆
- **过定方程组**：方程数 > 未知数，用伪逆求最优近似
- **欠定方程组**：未知数 > 方程数，伪逆给出最小范数解

### 关键公式

<div class="formula-box">

```
逆矩阵公式 (2×2)：
A⁻¹ = 1/(ad-bc) × [[d, -b], [-c, a]]

伪逆 (满列秩)：A⁺ = (AᵀA)⁻¹Aᵀ
伪逆 (满行秩)：A⁺ = Aᵀ(AAᵀ)⁻¹

线性回归：w = (XᵀX)⁻¹Xᵀy
```

</div>

### 思考题

为什么实际深度学习中很少直接用逆矩阵求解，而用梯度下降？

## Day 6: 特征值与特征向量

**一句话**：特征向量是变换的"不变方向"，特征值是缩放倍数。

### 核心定义

- **特征方程**：Av = λv
- **特征向量 v**：变换后方向不变（只缩放）
- **特征值 λ**：特征向量的缩放因子
- **特征分解**：A = VΛV⁻¹（可对角化时）

### 几何直观

<div class="callout callout-tip">

想象一个变换：大部分向量方向都变了，但特征向量方向保持不变，只被拉长或压缩。

- 特征值 > 1：拉长
- 0 < λ < 1：压缩
- λ < 0：反向

</div>

### AI 中的应用

- **PCA**：协方差矩阵的特征向量是主成分方向
- **谱聚类**：用拉普拉斯矩阵的特征向量做聚类
- **PageRank**：谷歌搜索的核心，转移矩阵的主特征向量
- **矩阵幂运算**：Aⁿ = VΛⁿV⁻¹，快速计算
- **稳定性分析**：特征值决定系统是否稳定

### 关键公式

<div class="formula-box">

```
特征方程：det(A - λI) = 0
特征分解：A = VΛV⁻¹
矩阵幂：Aⁿ = VΛⁿV⁻¹
迹与特征值：tr(A) = Σλᵢ
行列式与特征值：det(A) = Πλᵢ
```

</div>

## Day 7: 特征分解

**一句话**：将矩阵分解为特征向量基下的对角矩阵，简化计算。

### 核心定义

- **对角化**：A = VΛV⁻¹，V 是特征向量矩阵，Λ 是对角矩阵
- **可对角化条件**：n×n 矩阵有 n 个线性无关的特征向量
- **对称矩阵**：一定可对角化，且特征向量正交
- **谱定理**：实对称矩阵 = 正交矩阵 × 对角矩阵 × 正交矩阵转置

### AI 中的应用

- **PCA 计算**：协方差矩阵特征分解
- **二次型优化**：xᵀAx 在特征基下变成 Σλᵢyᵢ²
- **马尔可夫链**：稳态分布是转移矩阵的特征向量
- **图神经网络**：图拉普拉斯的特征分解用于谱卷积

### 关键公式

<div class="formula-box">

```
对称矩阵分解：A = QΛQᵀ (Q 是正交矩阵，Q⁻¹ = Qᵀ)
二次型：xᵀAx = yᵀΛy = Σλᵢyᵢ² (y = Qᵀx)
正定性：所有λᵢ > 0 ⇔ xᵀAx > 0 (∀x ≠ 0)
```

</div>

### 思考题

为什么对称矩阵的特征向量一定正交？这在 AI 中有什么用处？

## Day 8: 奇异值分解 SVD

**一句话**：SVD 是 AI 最重要的矩阵分解，适用于任意矩阵。

### 核心定义

- **SVD**：A = UΣVᵀ，对任意 m×n 矩阵都成立
- **U**：m×m 正交矩阵，左奇异向量（AAᵀ的特征向量）
- **Σ**：m×n 对角矩阵，奇异值σᵢ（从大到小排列）
- **V**：n×n 正交矩阵，右奇异向量（AᵀA 的特征向量）

### 几何直观

<div class="callout callout-tip">

任何矩阵分解 = 旋转 × 缩放 × 旋转

- Vᵀ：输入空间旋转
- Σ：各维度独立缩放（奇异值是缩放因子）
- U：输出空间旋转

</div>

### AI 中的应用

- **降维**：保留前 k 个奇异值，A ≈ UₖΣₖVₖᵀ
- **推荐系统**：矩阵分解，用户 - 物品矩阵分解为低秩近似
- **图像压缩**：用少量奇异值近似图像矩阵
- **LSA** (潜在语义分析)：词 - 文档矩阵 SVD 发现语义主题
- **去噪**：小奇异值对应噪声，截断后重构

### 关键公式

<div class="formula-box">

```
SVD 分解：A = UΣVᵀ
奇异值与特征值关系：σᵢ = √λᵢ(AᵀA)
低秩近似：Aₖ = Σᵢ₌₁ᵏ σᵢuᵢvᵢᵀ (最优 k 秩近似)
Frobenius 范数：||A||²_F = Σσᵢ²
```

</div>

### 思考题

SVD 和特征分解有什么区别？为什么 SVD 应用更广泛？

## Day 9: 主成分分析 PCA

**一句话**：PCA 用 SVD 找到数据方差最大的方向，实现降维。

### 核心定义

- **主成分**：数据方差最大的投影方向
- **第一主成分**：方差最大的方向
- **第二主成分**：与第一正交且方差次大的方向
- **累计方差贡献率**：前 k 个主成分解释的方差比例

### 几何直观

- **数据点云**：在高维空间中形成一个椭球
- **主成分**：椭球的长轴方向
- **投影**：将数据点投影到主成分张成的低维子空间
- **信息损失**：垂直于主成分方向的信息被丢弃

### AI 中的应用

- **数据可视化**：高维数据降到 2D/3D 可视化
- **特征提取**：提取主要特征，去除冗余
- **去噪**：小特征值方向对应噪声
- **预处理**：降维后输入神经网络，减少计算量
- **人脸识别**：特征脸 (Eigenface) 方法

### 关键公式

<div class="formula-box">

```
协方差矩阵：C = (1/n)XᵀX (X 已中心化)

PCA 步骤：
1. 数据中心化：X' = X - μ
2. 计算协方差：C = (1/n)X'ᵀX'
3. 特征分解：C = VΛVᵀ
4. 投影：Y = X'Vₖ (取前 k 个特征向量)

方差贡献率：λᵢ / Σλⱼ
```

</div>

### 思考题

为什么 PCA 要先对数据做中心化？不中心化会怎样？

## Day 10: 范数与距离

**一句话**：范数度量向量的"大小"，距离度量向量间的"远近"。

> **前置知识**：Day 1-2（向量与矩阵）  
> **后续应用**：Day 38（正则化详解）、Day 47（最小二乘）、Day 58（归一化）  
> **注意**：此处仅介绍范数概念，正则化的完整讲解在 Day 38

### 核心定义

| 范数 | 公式 | 名称 |
|------|------|------|
| L0 | 非零元素个数 | （不是真正的范数） |
| L1 | ||x||₁ = Σ|xᵢ| | 曼哈顿距离 |
| L2 | ||x||₂ = √(Σxᵢ²) | 欧氏距离 |
| L∞ | ||x||∞ = max|xᵢ| | 最大分量 |
| Lp | ||x||ₚ = (Σ|xᵢ|ᵖ)^(1/p) | p-范数 |

### 几何直观

- **L2 范数**：从原点的直线距离（圆形等值线）
- **L1 范数**：沿坐标轴走的距离（菱形等值线）
- **L∞ 范数**：最大坐标的绝对值（方形等值线）
- **稀疏性**：L1 更容易产生稀疏解（坐标轴方向的尖角）

### AI 中的应用

- **正则化**：L1 (Lasso) 产生稀疏性，L2 (Ridge) 防止过拟合
- **损失函数**：MSE 用 L2，MAE 用 L1
- **归一化**：BatchNorm/LayerNorm 用 L2 范数
- **相似度**：余弦相似度 = 归一化后的点积
- **聚类**：KMeans 用 L2 距离，KMedians 用 L1 距离

### 关键公式

<div class="formula-box">

```
L1 范数：||x||₁ = Σ|xᵢ|
L2 范数：||x||₂ = √(Σxᵢ²) = √(x·x)
Lp 范数：||x||ₚ = (Σ|xᵢ|ᵖ)^(1/p)

范数不等式：||x||∞ ≤ ||x||₂ ≤ ||x||₁ ≤ √n·||x||₂

L1 正则化：L = Loss + λ||w||₁
L2 正则化：L = Loss + λ||w||₂²
```

</div>

### 思考题

为什么 L1 正则化能产生稀疏解，而 L2 不能？从几何角度解释。
