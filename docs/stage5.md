## Day 41: 信息量与自信息

**一句话**：信息量度量事件的"surprise 程度"，越不可能的事件信息量越大。

### 核心定义

- **自信息**：I(x) = -log P(x) = log(1/P(x))
- **单位**：bit（以 2 为底）或 nat（以 e 为底）
- **性质**：
  - P(x)=1 时，I(x)=0（确定事件无信息）
  - P(x)→0 时，I(x)→∞（罕见事件信息量大）

### 关键公式

<div class="formula-box">

```
I(x) = -log₂ P(x)  (单位：bit)
I(x) = -ln P(x)    (单位：nat)
独立事件：I(x,y) = I(x) + I(y)
```

</div>

## Day 42: 熵 Entropy

**一句话**：熵是随机变量的平均信息量，度量不确定性。

### 核心定义

- **熵**：H(X) = -Σ P(x)log P(x) = E[I(X)]
- **最大熵**：均匀分布时熵最大
- **最小熵**：确定性分布时熵为 0

### 关键公式

<div class="formula-box">

```
离散熵：H(X) = -Σᵢ P(xᵢ)log₂ P(xᵢ)
二元熵：H(p) = -p log₂ p - (1-p)log₂(1-p)
最大值：p=0.5 时 H=1 bit
```

</div>

## Day 43: 交叉熵

**一句话**：交叉熵度量用错误分布 q 编码真实分布 p 的平均编码长度。

### 核心定义

- **交叉熵**：H(p,q) = -Σ p(x)log q(x)
- **与熵关系**：H(p,q) = H(p) + D_KL(p||q)
- **最小化交叉熵** = 最小化 KL 散度

### AI 应用

- 分类损失函数
- softmax + 交叉熵
- 语言模型

### 关键公式

<div class="formula-box">

```
H(p,q) = -Σₓ p(x)log q(x)
分类问题：L = -log q(y_true)
多分类：L = -Σᵢ Σₖ yᵢₖ log ŷᵢₖ
```

</div>

## Day 44: KL 散度

**一句话**：KL 散度度量两个概率分布的"距离"（非对称）。

### 核心定义

- **KL 散度**：D_KL(p||q) = Σ p(x)log(p(x)/q(x))
- **性质**：
  - D_KL(p||q) ≥ 0
  - D_KL(p||q) = 0 当且仅当 p=q
  - 非对称：D_KL(p||q) ≠ D_KL(q||p)

### AI 应用

- 变分推断
- VAE
- 知识蒸馏

### 关键公式

<div class="formula-box">

```
D_KL(p||q) = Σₓ p(x)log(p(x)/q(x))
= Eₓ~p[log p(x) - log q(x)]
D_KL(p||q) = H(p,q) - H(p)
```

</div>

---

## Day 45: 互信息

**一句话**：互信息度量两个变量"共享"的信息量。

### 核心定义

- **互信息**：I(X;Y) = ΣΣ P(x,y)log(P(x,y)/(P(x)P(y)))
- **等价形式**：
  - I(X;Y) = H(X) - H(X|Y)
  - I(X;Y) = H(Y) - H(Y|X)
  - I(X;Y) = D_KL(P(X,Y) || P(X)P(Y))

### AI 应用

- 特征选择
- 表示学习
- 对比学习 (InfoNCE)

### 关键公式

<div class="formula-box">

```
I(X;Y) = H(X) - H(X|Y)
= H(X) + H(Y) - H(X,Y)
= D_KL(P(X,Y) || P(X)P(Y))
```

</div>
