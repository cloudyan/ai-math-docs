## 阶段 3：概率论基础（Day 21-30）

_理解不确定性和随机性_

### Day 21: 概率与条件概率

**一句话**：概率度量事件发生的可能性，条件概率是"已知某信息后"的概率。

**核心公式**：
<div class="formula-box">

```
条件概率：P(A|B) = P(A∩B)/P(B), P(B) > 0
乘法法则：P(A∩B) = P(A|B)P(B) = P(B|A)P(A)
全概率公式：P(A) = ΣᵢP(A|Bᵢ)P(Bᵢ)
```

</div>

**AI 应用**：分类问题、贝叶斯网络、语言模型

---

### Day 22: 贝叶斯定理

**一句话**：贝叶斯定理是"逆向概率"，从结果反推原因。

**核心公式**：
<div class="formula-box">

```
贝叶斯定理：P(A|B) = P(B|A)P(A)/P(B)
```

</div>

**AI 应用**：朴素贝叶斯、贝叶斯推断、贝叶斯优化

---

### Day 23: 随机变量

**一句话**：随机变量是将随机结果映射为数字的函数。

**核心公式**：
<div class="formula-box">

```
离散 PMF: P(X=xᵢ) = pᵢ, Σpᵢ = 1
连续 PDF: p(x) ≥ 0, ∫p(x)dx = 1
CDF: F(x) = P(X ≤ x)
```

</div>

**AI 应用**：分类/回归输出、生成模型、Dropout

---

### Day 24: 概率分布

**一句话**：常见概率分布是 AI 建模的基础构件。

**核心分布**：

| 分布 | 公式 | 应用 |
|------|------|------|
| 伯努利 | P(X=1) = p | 二分类 |
| 二项 | C(n,k)pᵏ(1-p)ⁿ⁻ᵏ | n 次试验 |
| 高斯 | (1/√(2πσ²))exp(-(x-μ)²/(2σ²)) | 回归误差 |

---

### Day 25: 期望与方差

**一句话**：期望是随机变量的"平均值"，方差度量"波动程度"。

**核心公式**：
<div class="formula-box">

```
期望：E[X] = ΣxᵢP(X=xᵢ) (离散) / ∫x·p(x)dx (连续)
方差：Var(X) = E[X²] - (E[X])²
```

</div>

**AI 应用**：损失函数、强化学习、批归一化、权重初始化

## 阶段 4：统计与推断（Day 31-40）

_从数据中学习参数和做出推断_

### Day 31: 最大似然估计 MLE

**一句话**：MLE 选择让观测数据"最可能出现"的参数值。

**核心公式**：
<div class="formula-box">

```
似然：L(θ|D) = P(D|θ) = ΠᵢP(xᵢ|θ)
MLE: θ̂ = argmax ℓ(θ) = argmax Σlog P(xᵢ|θ)
```

</div>

**AI 应用**：线性回归、逻辑回归、语言模型

---

### Day 32: 最大后验估计 MAP

**一句话**：MAP 在 MLE 基础上加入先验，是"正则化的 MLE"。

**核心公式**：
<div class="formula-box">

```
MAP: θ̂ = argmax [log P(D|θ) + log P(θ)]
高斯先验 → L2 正则化
拉普拉斯先验 → L1 正则化
```

</div>

</div>
